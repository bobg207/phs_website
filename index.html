Portland High School

Mr Girsch's Computer Science Classes

Portland High School is excited to offer 4 distinct Computer Science courses:

  • Intro to Coding with Python
  • Data Analysis
  • Web Development
  • Game Development with Unity

Each course provides students experiences in design and development.

Intro to Coding with Python

Learn basic computer programming concepts and terminologies by completing hands-on exercises.

Data Analysis

Analyze large datasets using Python's Pandas library and create visualizations with Matplotlib.

Web Development

Learn UI/UX principles to design attractive sites. Then create the front-end using HTML5, CSS3, and JavaScript.

Game Development

Design, animate, and code, using C#, games and/or experiences in both 2D and 3D.
